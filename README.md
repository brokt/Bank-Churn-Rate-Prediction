# Bank Churn Rate Prediction
Predicting if a customer is likely to leave a bank. 

## Input Format
Input layer neurons(12)
* 0 - France 0 or 1
* 1 - Spain 0 or 1
* 2 - Germany 0 or 1
* 3 - CreditScore
* 4 - Gender 0 -Female and 1 Male
* 5 - Age
* 6 - Tenure
* 7 - Balance
* 8 - NumOfProducts
* 9 - HasCrCard
* 10 - IsActiveMember
* 11 - EstimatedSalary


## Dependencies
1) Tensorflow
2) Keras
3) numpy
4) pandas
5) sklearn
